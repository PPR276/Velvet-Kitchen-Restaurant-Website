# 🍽️ Velvet Kitchen – Multi-Page Restaurant Website

A multi-page restaurant website developed as part of a Frontend Capstone Project.  
The project simulates a complete restaurant web experience including menu browsing, ordering, reservations, reviews, login system, and feedback.

---

## 📌 Project Overview

Velvet Kitchen is a fully designed restaurant website built using HTML, CSS, and JavaScript.  

The application provides users with:

- Homepage introduction
- Menu browsing with filters and slideshow
- Online ordering system
- Table reservation form
- Customer review system
- Login/Sign-up UI
- Order tracking dashboard
- Feedback submission form
- Team showcase page

This project demonstrates frontend development concepts and UI structuring across multiple interconnected pages.

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Responsive Design Principles

---

## 📂 Project Structure

Main Pages:

- `velvetkitchen.html` → Homepage (Main Entry Page)
- `menu.html` → Menu with filters and slideshow
- `signin.html` → Login / Sign-up UI
- `reservation.html` → Table reservation form
- `orderpagefinal.html` → Order placement system
- `orders.html` → Order tracking dashboard
- `Review.html` → Customer review system
- `feedback.html` → Feedback submission form
- `Team.html` → Team members page

Supporting pages:
- Payment success page
- Delivery confirmation page
- Contact page
